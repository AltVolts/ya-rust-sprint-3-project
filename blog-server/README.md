# Backend часть приложения блога

## Серверная часть приложения, построенная на Rust фреймворке actix-web. Взаимодействие возможно как по HTTP, так и по grpc протоколам


### Основная часть настроек берется из .env файл
### Для взаимодействия с базой данных на postgres используется sqlx, миграции применяются на старте приложения и расположены в директории /migrations
