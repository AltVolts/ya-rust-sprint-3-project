# Wasm frontend-приложение для блога на Yew

## Web приложение, построенное на Rust фреймворке Yew. Взаимодействует с основным сервером по HTTP.
### Ожидает, что основной url для взаимодействия с сервером - http://localhost:3000/api
### Для раздачи сгенерированных статических файлов используется trunk, сами файлы будут доступны в директории /dist
